SEO-optimized landing page for Shizuku Matcha Studio, a premium matcha tea business in Hermosillo, Sonora. Built with Next.js and Tailwind CSS, integrated with Strapi CMS for flexible content management. Frontend deployed on Vercel, backend on Railway. Features product showcase, origin story from Shizuoka, Japan, and wholesale information. Built to rank #1-2 for "Matcha ceremonial Hermosillo" and capture local buyers searching for authentic Japanese ceremonial matcha.
